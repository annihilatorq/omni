## Constants

`omni::mem` содержит allocation flags:

- `commit`
- `reserve`
- `commit_reserve`
- `large_commit`
- `release`

`omni::mem::page` содержит page-protection flags:

- `no_access`
- `read_only`
- `read_write`
- `execute`
- `execute_read`
- `execute_read_write`
- `guard`
- `no_cache`
- `write_combine`

## `omni::nt_allocator<T, AllocFlags, Protect>`

Standard allocator-compatible type на базе `NtAllocateVirtualMemory` и `NtFreeVirtualMemory`.

| Method | Purpose |
| --- | --- |
| `allocate(n)` | выделить storage для `n` elements |
| `allocate(address, n)` | запросить allocation по конкретному address |
| `deallocate(ptr, n)` | освободить storage |

Allocator предоставляет usual allocator typedefs, `rebind`, equality operators и `is_always_equal`. При allocation failure он бросает `std::bad_alloc`, если exceptions включены; иначе завершает процесс.

## Aliases

- `omni::rw_allocator<T>` — read/write memory
- `omni::rx_allocator<T>` — execute/read memory
- `omni::rwx_allocator<T>` — execute/read/write memory

## Пример

```cpp
auto allocator = omni::rw_allocator<std::byte>{};
auto* page = allocator.allocate(4096);
auto view = std::span<std::byte>{page, 4096};

std::ranges::fill(view, std::byte{0});
allocator.deallocate(page, 4096);
```
