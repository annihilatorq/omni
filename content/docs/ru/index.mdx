# omni

`omni` — header-only C++23 библиотека для Windows-oriented low-level разработки: inspection модулей и экспортов, чтение API-set schema, lazy import, syscall wrappers, shared user data, allocators и compile-time hashing.

## Основные возможности

- доступ к loader list как к range через `omni::modules`;
- разбор PE export table через `omni::module_exports` и `omni::module_export`;
- чтение API-set schema через `omni::api_sets` и `omni::api_set`;
- вызов WinAPI через `omni::lazy_import` и `omni::lazy_importer`;
- вызов x64 NT syscalls через `omni::syscall` и `omni::syscaller`;
- доступ к `KUSER_SHARED_DATA` через `omni::shared_user_data()`;
- строго типизированные compile-time hashes с заменяемой hash policy;
- optional caching для lazy imports и syscall IDs.
