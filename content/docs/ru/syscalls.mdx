## Platform limitation

`omni::syscaller` и `omni::syscall` доступны только на x64.

## `omni::syscaller<T>`

Typed wrapper находит export, извлекает syscall ID из stub, подготавливает shellcode bridge и выполняет system call.

### Constructor

- `syscaller(export_name_hash, syscall_id_parser parser = default_syscall_id_parser)`
- `syscaller(default_hash, syscall_id_parser parser = default_syscall_id_parser)`

`syscaller<T>::syscall_id_parser` — тип parser. Parser по умолчанию: `omni::default_syscall_id_parser(const omni::module_export&)`.

### Methods

| Method | Returns | Purpose |
| --- | --- | --- |
| `try_invoke(args...)` | `std::expected<T, std::error_code>` | вызов с диагностикой |
| `invoke(args...)` | `T` | вызов с default value при ошибке |
| `operator()(args...)` | `T` | shorthand для `invoke` |

Function-pointer specializations выводят arguments и return type из NT function signature.

## `omni::syscall`

Free overloads дают короткую запись для one-off calls:

- `omni::syscall<T = omni::status>(export_name_hash, args...)`
- `omni::syscall<F>(export_name_hash, args...)`
- `omni::syscall<Func, Hasher>(args...)`
- `omni::syscall<Func>(args...)`

## Примеры

```cpp
auto yield_status = omni::syscall<omni::status>("NtYieldExecution");
```

```cpp
using nt_query_info_process_fn = omni::status (*)(HANDLE, ULONG, void*, ULONG, ULONG*);

omni::syscaller<nt_query_info_process_fn> query_process{"NtQueryInformationProcess"};
auto result = query_process.try_invoke(
  ::GetCurrentProcess(),
  0U,
  &process_info,
  sizeof(process_info),
  &return_length
);
```

```cpp
auto status = omni::syscall<nt_query_info_process_fn>(
  "NtQueryInformationProcess",
  ::GetCurrentProcess(),
  0U,
  &process_info,
  sizeof(process_info),
  &return_length
);
```

## Errors and notes

`try_invoke()` может вернуть `export_not_found`, `module_not_loaded` или `syscall_id_not_found`. Последняя ошибка означает, что export найден, но не совпал с ожидаемым syscall stub pattern.

Free overloads без `try_*` возвращают default-constructed value при ошибке. Default parser не обрабатывает hooks или EDR-modified stubs.
