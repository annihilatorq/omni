## Enumerations

### `omni::severity`

Классифицирует high bits значения NTSTATUS:

- `success`
- `information`
- `warning`
- `error`

### `omni::facility`

Представляет NT status facility/subsystem. Enum включает common groups вроде `rpc_runtime`, `io_error_code`, `ntwin32`, `graphics_kernel`, `hypervisor` и `security_core`.

### `omni::ntstatus`

Содержит часто используемые NTSTATUS constants, включая:

- success и informational states: `success`, `pending`, `more_entries`
- iteration end states: `no_more_entries`, `no_more_files`
- validation и lookup errors: `invalid_parameter`, `object_name_not_found`, `procedure_not_found`
- access и memory errors: `access_denied`, `access_violation`, `no_memory`
- module и feature errors: `dll_not_found`, `not_implemented`, `not_supported`, `unsuccessful`

## `omni::status`

Lightweight wrapper над raw `NTSTATUS`.

| API | Назначение |
| --- | --- |
| `value` | raw `std::int32_t` status value |
| `operator=(std::int32_t)` | присвоить raw value |
| `operator=(omni::ntstatus)` | присвоить из enum |
| `is_success()` | проверить non-negative status |
| `is_information()` | проверить information severity |
| `is_warning()` | проверить warning severity |
| `is_error()` | проверить error severity |
| `severity()` | вернуть `omni::severity` |
| `facility()` | вернуть `omni::facility` |
| `code()` | вернуть low 16-bit code |

Тип поддерживает comparisons с raw integers, `omni::ntstatus` и другими `omni::status`. Также доступны explicit conversion to `bool` и `std::int32_t`, плюс `std::formatter` для `omni::status` и `omni::ntstatus`.

## Пример

```cpp
omni::status result = omni::ntstatus::success;

if (result.is_success()) {
  std::println("status: 0x{:08X}", static_cast<std::uint32_t>(result.value));
}

if (result == omni::ntstatus::success) {
  std::println("exact success");
}
```
