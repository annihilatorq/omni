# omni

`omni` is a header-only C++23 library for Windows-focused low-level work: module and export inspection, API-set schema reading, lazy imports, syscalls, shared user data, allocators, and compile-time hashing.

## Highlights

- loader-list access through range-friendly `omni::modules`;
- PE export table parsing through `omni::module_exports` and `omni::module_export`;
- API-set schema inspection through `omni::api_sets` and `omni::api_set`;
- WinAPI calls through `omni::lazy_import` and `omni::lazy_importer`;
- x64 NT syscall calls through `omni::syscall` and `omni::syscaller`;
- `KUSER_SHARED_DATA` access through `omni::shared_user_data()`;
- strongly typed compile-time hashes with replaceable hash policies;
- optional internal caching for lazy imports and syscall IDs.

## Scope

This site documents the public user-facing API. Headers under `omni/detail/*` are implementation details and are not treated as a supported contract.
