## `omni::address`

`omni::address` is a small wrapper over a memory address.

| API | Purpose |
| --- | --- |
| constructors | from `nullptr`, `std::uintptr_t`, pointer, or contiguous range |
| `ptr<T>(offset = 0)` | return a typed pointer at an offset |
| `value()` | return the raw `std::uintptr_t` |
| `offset<T>(delta = 0)` | produce an offset address or pointer |
| `as<T>()` | cast the address to another type |
| `span<T>(count)` | view memory as `std::span` |
| `is_in_range(start, end)` | check whether the address is inside a range |
| `invoke<T>(args...)` | call the address as a function |

It supports explicit boolean conversion, comparisons, arithmetic and bitwise operators, stream output, and `std::formatter<omni::address>`.

```cpp
omni::address base{::GetModuleHandleW(nullptr)};
auto* image = base.ptr<void>();
auto entry = base.offset(0x1000);
```

## Hash API

### Types and functions

- `omni::fnv1a32`
- `omni::fnv1a64`
- `omni::default_hash`, an alias for `omni::fnv1a64`
- `omni::hash_pair<Hasher = default_hash>`
- `omni::hash<Hasher>(value)`

Hashes are ASCII case-insensitive, which makes them suitable for Windows module and export names.

```cpp
constexpr omni::fnv1a64 kernel32{"kernel32.dll"};
auto export_hash = omni::hash<omni::fnv1a32>("GetCurrentProcessId");
constexpr omni::hash_pair<> lookup{"GetSystemDirectoryW", "kernel32.dll"};
```

### Literals

The `omni::literals` namespace provides string literal operators for narrow and wide strings:

- `"..."_fnv1a32`
- `"..."_fnv1a64`
- `"..."_hash`

### Formatting

`std::formatter` support is available for `omni::fnv1a32` and `omni::fnv1a64`.

## Custom hash policies

Most hash-aware APIs are templated on `Hasher`. The same policy can be used with modules, exports, lazy imports, and syscalls.

A hasher must satisfy `omni::concepts::hash`: it needs a `value_type`, `value()`, comparisons, construction from strings/values, and hashing for supported string or span inputs.

```cpp
using my_hash = omni::fnv1a32;

auto module = omni::get_module(my_hash{L"kernel32.dll"});
auto export_entry = omni::get_export(my_hash{"GetCurrentProcessId"}, my_hash{"kernel32.dll"});
auto result = omni::lazy_import<void, my_hash>(my_hash{"SetLastError"}, 0U);
```

## Concepts

Public concepts live in `omni/concepts.hpp`:

- `omni::concepts::hash`
- `omni::concepts::hashable`
- `omni::concepts::function_pointer`

## Error API

`enum class omni::error` values:

- `module_not_loaded`
- `export_not_found`
- `syscall_id_not_found`
- `forwarder_string_invalid`

The errors integrate with `std::error_code` through `omni::error_category()`, `omni::make_error_code()`, and the standard `std::is_error_code_enum` specialization.

```cpp
std::error_code ec = omni::error::export_not_found;
```
