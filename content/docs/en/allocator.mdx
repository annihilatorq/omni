## Constants

`omni::mem` contains allocation flags:

- `commit`
- `reserve`
- `commit_reserve`
- `large_commit`
- `release`

`omni::mem::page` contains page-protection flags:

- `no_access`
- `read_only`
- `read_write`
- `execute`
- `execute_read`
- `execute_read_write`
- `guard`
- `no_cache`
- `write_combine`

## `omni::nt_allocator<T, AllocFlags, Protect>`

A standard allocator-compatible type backed by `NtAllocateVirtualMemory` and `NtFreeVirtualMemory`.

| Method | Purpose |
| --- | --- |
| `allocate(n)` | allocate storage for `n` elements |
| `allocate(address, n)` | request allocation at a specific address |
| `deallocate(ptr, n)` | release storage |

The allocator exposes the usual allocator typedefs, `rebind`, equality operators, and `is_always_equal`. On allocation failure it throws `std::bad_alloc` when exceptions are enabled; otherwise it terminates.

## Aliases

- `omni::rw_allocator<T>` — read/write memory
- `omni::rx_allocator<T>` — execute/read memory
- `omni::rwx_allocator<T>` — execute/read/write memory

## Example

```cpp
auto allocator = omni::rw_allocator<std::byte>{};
auto* page = allocator.allocate(4096);
auto view = std::span<std::byte>{page, 4096};

std::ranges::fill(view, std::byte{0});
allocator.deallocate(page, 4096);
```
